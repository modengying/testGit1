# C#程序的结构

p

# 1.1 using 关键字

using 关键字 后面添加名命空间，表示可以省略该命名空间直接使用里面的程序内容。

# 1.2 namespace 命名空间声明，

使用namespace 来声明一个命名空间，空间名称可以自定义，但不能是中文或关键字，特殊符号也不建议使用

# 1.2.1 class 类声明

一个命名空间中可以声明多个类，类中可以声明 字段，属性，方法，函数。

# 1.2.2 Main()方法

Main方法是C#运行程序的入口，其中可以包含语句和表达式，调用方法和函数。
