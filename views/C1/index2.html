# C# 基本语法

# 2.1数据类型

数据类型 常用的有两种，一种是值类型，如 int , char ,float等 ；另一种是引用类型 如string object,Array等 值类型 是直接在栈中声明，也储存栈中，；值类型好比货币，直接拿在手里，买东西时可以直接交易。 引用类型 在栈中声明， 储存在堆，栈中储存的是其的引用；引用类型好比银行存折，买东西时要去银行取钱才行。

# 2.2数据类型的转换

数据类型转换就是把一种数据类型转换为另一种类型，C#中有两种转换。 隐式转换-- C#中默认的安全转换，不会存在数据丢失的情况，比如值范围小的类型转换为大的类型。

显式强制转换--显式转换需要有强制转换的运算符，而且强制转换可能会造成数据丢失。
